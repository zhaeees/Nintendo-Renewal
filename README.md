## 📌 Project Overview

기존 닌텐도 웹사이트를 분석하여  
콘텐츠 구조와 UI 흐름을 개선한 웹사이트 리뉴얼 팀 프로젝트입니다.

사용자 경험을 중심으로 정보 위계를 재정리하고,  
퍼블리싱 관점에서 완성도 높은 UI 구현을 목표로 진행했습니다.

<br>

## 🎯 Project Goal

- 기존 닌텐도 웹사이트의 정보 구조와 UI 흐름을 분석하여 **사용자 경험을 개선**
- 주요 콘텐츠가 명확하게 전달되도록 **콘텐츠 위계와 레이아웃 재구성**

- 시맨틱 마크업과 일관된 스타일링을 적용하여 **퍼블리싱 완성도 향상**

## 📝 What I Learned


- 기존 웹사이트를 분석하고 리뉴얼 방향을 설정하는 과정에서 **콘텐츠 구조와 사용자 흐름의 중요성**을 이해하게 되었습니다.
- 팀 프로젝트를 통해 역할 분담과 협업의 필요성을 경험하며, **GitHub와 Slack을 활용한 협업 흐름**에 익숙해졌습니다.
- SCSS 구조화와 페이지별 스타일 분리를 통해 **유지보수를 고려한 퍼블리싱 방식**을 학습했습니다.
- 문제 해결 과정에서 AI 도구를 보조적으로 활용하며 **스스로 원인을 분석하고 해결 방향을 판단하는 역량**의 중요성을 느꼈습니다.


- 가로 스크롤 구현 과정에서 발생한 다양한 오류를 수정하며, **가로 스크롤 구조와 이벤트 흐름**을 체계적으로 이해하게 되었습니다.
- 자동 재생 애니메이션 구현 과정에서 발생한 반복 끊김과 위치 오차 문제를 직접 수정하며, 무한 루프 구조와 타이밍 제어 로직에 대한 이해도를 높였습니다. 이를 바탕으로, 추후 다양한 UI 상황에 적용하며 인터랙션 활용 범위를 지속적으로 확장해 나갈 계획입니다.
- 아코디언 UI 구현 시 JavaScript로 직접 열고 닫는 로직만 사용해 왔지만, **`<details>`와 `<summary>` 태그를 활용한 시맨틱한 구현 방식**을 알게 되면서 상황에 따라 더 간결하고 의미 있는 마크업을 선택할 수 있게 되었습니다.

## ⚠️ Trouble Shooting

### 1. 배포 환경에서 이미지가 정상적으로 로딩되지 않는 문제

**🔍 문제 상황**  
GitHub Pages로 배포한 이후, 로컬 환경에서는 정상적으로 보이던 이미지가 배포 환경에서 깨져 보이는 문제가 발생했습니다.

**✅ 해결 방법**  
이미지 경로가 절대 경로로 작성되어 있던 것을 확인하고, 모든 이미지 경로를 **상대 경로(`./`) 기준으로 수정**하여 배포 환경에서도 이미지가 정상적으로 로딩되도록 개선했습니다.

---

### 2. JavaScript 코드 병합 과정에서 발생한 스코프 충돌 문제

**🔍 문제 상황**  
팀원들의 JavaScript 코드를 병합하는 과정에서 전역 스코프에 선언된 변수와 함수가 서로 충돌하여 일부 기능이 정상적으로 동작하지 않는 문제가 반복적으로 발생했습니다.

**✅ 해결 방법**  
각 기능을 **즉시 실행 함수(IIFE)**로 감싸 스코프를 분리하고, 기능 단위로 코드를 정리하여 서로 간섭 없이 독립적으로 동작하도록 구조를 개선했습니다.

---

### 3. 프로젝트 전반의 자잘한 오류 및 코드 안정성 문제

**🔍 문제 상황**  
프로젝트 진행 중 콘솔 에러와 예기치 않은 동작 등 자잘한 오류가 누적되어 코드 안정성이 저하되는 문제가 있었습니다.

**✅ 해결 방법**  
콘솔 로그를 기준으로 오류 원인을 하나씩 점검하며 조건문, 이벤트 처리, 변수 사용 방식 등을 수정하여 **전체적인 코드 안정성과 완성도를 향상**시켰습니다.

---

### 4. 동영상 파일 배포 과정에서 발생한 업로드 문제

**🔍 문제 상황**  
동영상 파일을 이미지와 동일하게 `images/` 폴더에 포함하여 관리하고 GitHub Pages를 통해 배포하려 했으나, cmd를 통한 업로드 과정에서 동영상 파일이 정상적으로 업로드되지 않는 문제가 발생했습니다.

**✅ 해결 방법**  
GitHub Pages 환경에서 대용량 미디어 파일 관리에 제약이 있다는 점을 확인하고, 동영상 파일을 외부 호스팅 서비스(닷홈)에 별도로 업로드한 뒤 `video` 태그의 경로를 외부 URL로 분리하여 정상적으로 재생되도록 해결했습니다.
